# Skill 名称：社交媒体视频情报提取 (Social Media Video Intel Extraction)

## 适用场景
当需要从社交媒体平台（如小红书、抖音、TikTok）的大量短视频中提取结构化知识、分析行业趋势、总结操作教程或进行竞品调研时使用。适用于需要处理非结构化视频数据并将其转化为高价值文档的场景。

## 步骤
1. **关键词定义与搜索**：
   - 确定核心关键词及长尾词。
   - 使用搜索工具获取视频元数据（标题、描述、点赞数、评论数、视频链接）。
2. **筛选与过滤**：
   - 根据互动率（点赞/收藏比）和相关度过滤前 5-10 个高质量视频。
   - 排除纯广告或低质量搬运内容。
3. **多模态数据抓取**：
   - 批量下载视频文件或提取音频流。
   - 抓取视频评论区内容以获取补充信息（如：避雷点、补充步骤）。
4. **内容解析（多模态分析）**：
   - **ASR (语音转文字)**：提取视频中的口播内容。
   - **OCR (视觉文字识别)**：提取视频画面中的字幕、PPT、关键步骤文字。
   - **关键帧分析**：识别视频中的演示动作或实物展示。
5. **交叉比对与知识聚合**：
   - 对比多个视频的观点，识别共性知识（核心原理）和特性知识（独特技巧）。
   - 验证信息的准确性，标记冲突点。
6. **报告生成**：
   - 按照逻辑结构（如：背景、步骤、注意事项、工具推荐）组织内容。
   - 使用 Markdown 编写并可选导出为 PDF。

## 工具和命令
- **搜索与下载**：使用 `media-downloader` 或自定义脚本调用平台 API。
  ```bash
  # 示例：下载指定 URL 列表的视频
  python download_manager.py --urls list.txt --output ./raw_videos/
  ```
- **音频提取与转录**：使用 `ffmpeg` 和 `Whisper`。
  ```bash
  # 提取音频
  ffmpeg -i video.mp4 -q:a 0 -map a output.mp3
  # 语音转文字
  whisper output.mp3 --model medium --language Chinese
  ```
- **视觉提取**：使用 `PaddleOCR` 或 `Tesseract`。
- **聚合分析**：调用 LLM (如 GPT-4o) 进行多文本总结。

## 注意事项
- **反爬虫机制**：频繁请求会导致 IP 被封禁，建议设置合理的延迟（10-30s）或使用代理。
- **口语化处理**：短视频口播通常包含大量冗余词（如“那么”、“然后”），在聚合阶段需进行去噪处理。
- **时效性**：社交媒体内容更新极快，需在报告中注明数据抓取的时间戳。
- **版权合规**：提取的内容仅用于个人学习或内部分析，严禁直接洗稿发布。

## 示例
**任务**：提取小红书上关于“自制酸梅汤”的专业配方。
1. **搜索**：关键词“酸梅汤 配方 比例”。
2. **筛选**：选取点赞过万的 5 个不同博主的视频。
3. **解析**：
   - 博主 A 强调乌梅必须是烟熏的。
   - 博主 B 提供了精确到克的配比（乌梅30g, 山楂25g...）。
   - 博主 C 提示了熬煮前需浸泡 30 分钟。
4. **聚合**：生成一份包含“标准配方表”、“关键工艺（浸泡、火候）”、“进阶技巧（烟熏风味）”的 Markdown 文档。
5. **输出**：`SuanMeiTang_Recipe_Intel.md`。
